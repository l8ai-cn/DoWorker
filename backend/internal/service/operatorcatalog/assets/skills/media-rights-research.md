# Media Rights Research

Find usable media while preserving source and license evidence.

## Workflow

1. Translate the shot list into concrete asset queries.
2. Prefer user-owned media, official product media, public-domain archives, or assets with explicit commercial reuse terms.
3. Record the source URL, creator, license, retrieval date, required attribution, and modification restrictions for every selected asset.
4. Reject results whose license cannot be verified from the source.
5. Download the highest useful quality without upscaling low-resolution previews.
6. Store an asset manifest beside the project and map each file to its intended shot.

Do not treat search-engine visibility, repository availability, or model output as proof of redistribution rights.
